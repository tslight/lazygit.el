* Git Hosting API Clients for Emacs

A very basic Emacs client for GitLab and GitHub so far...

Get slack - be a lazy git!

** Example Use Package Configuration

#+begin_src emacs-lisp
(use-package lazygit
  :load-path "~/src/gitlab/tspub/lisp/lazygit")

(use-package gitlab
  :after lazygit
  :bind
  ("C-c C-g l c" . gitlab/clone-or-pull-project)
  ("C-c C-g l g" . gitlab/clone-or-pull-group)
  ("C-c C-g l r" . gitlab/retriever)
  :config
  (setq gitlab/token-file "~/.gitlab.key"))

(use-package github
  :after lazygit
  :bind
  ("C-c C-g h a" . github/clone-or-pull-all)
  ("C-c C-g h c" . github/clone-or-pull-repo)
  ("C-c C-g h r" . github/retriever)
  :config
  (setq github/token-file "~/.github.key"))
#+end_src
